## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 77486](https://school.programmers.co.kr/learn/courses/30/lessons/77486)
- 문제 요약: 판매원이 칫솔을 판매했을 때 이익의 10%를 추천인에게 분배한 뒤 각 판매원의 최종 이익을 구하는 문제.

## 📌 풀이 방법
1. enroll과 referral을 이용해 각 판매원의 추천인을 Map에 저장한다. 
2. 각 판매원의 누적 이익을 저장할 Map을 0으로 초기화한다. 
3. seller와 amount를 순회하며 판매 금액을 계산한다. 
4. 현재 판매자는 금액의 90%를 가지고, 10%는 추천인에게 넘긴다. 
5. 추천인이 없거나 넘길 금액이 0원이 될 때까지 위 과정을 반복한다. 
6. enroll 순서대로 각 판매원의 최종 이익을 배열에 담아 반환한다.

## ✅ 느낀 점
조직도를 직접 트리로 만들기보다 Map으로 추천인 관계를 저장하면 이익 분배를 간단하게 처리할 수 있었다.
