## 💡 백준 문제
- 문제 링크: [백준 11650](https://www.acmicpc.net/problem/11650)
- 문제 요약: N개의 점을 x 오름차순, x가 같으면 y 오름차순으로 정렬해서 출력하는 문제

## 📌 풀이 방법
람다식을 사용해서 풀이를 했다. <br>
Array.sort는 2차원배열을 정렬하지 못하기에 
Array.sort()을 활용해서 arr -> (a1, a2)로 정렬해줬다. <br>
만약 a1[0] == a2[0]가 같으면 return a1[1] - a2[1];로 결과값이 음수면 앞, 양수면 뒤로 배치된다는 Comparator 규칙을 이용해 정렬했다.

## ✅ 느낀 점
람다식을 사용하니 복잡한 비교 로직을 직접 작성할 필요 없이
Arrays.sort()만으로 간단하게 정렬 기준을 정의할 수 있었다.
