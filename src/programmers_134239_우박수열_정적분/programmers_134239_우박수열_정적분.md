## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 134239](https://school.programmers.co.kr/learn/courses/30/lessons/134239)
- 문제 요약: 콜라츠(우박) 수열 그래프에서 주어진 구간들의 정적분 값을 구하는 문제.

## 📌 풀이 방법
1. k부터 시작해 우박수열을 생성하면서 인접 값마다 사다리꼴 넓이 (y1 + y2)/2를 리스트에 저장한다.
2. 각 range에 대해 시작 인덱스 x와 끝 인덱스 y(= 전체 길이 + offset)를 계산한다. 
3. x > y이면 -1, x == y이면 0, x < y이면 해당 구간의 넓이를 누적합으로 계산해 반환한다.

## ✅ 느낀 점
우박수열 생성보다 구간 합을 효율적으로 계산하는 방식이 핵심이었다.
