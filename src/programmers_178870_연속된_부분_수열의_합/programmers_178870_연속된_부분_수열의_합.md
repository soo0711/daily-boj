## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 178870](https://school.programmers.co.kr/learn/courses/30/lessons/178870)
- 문제 요약: 연속 부분 수열의 합이 k가 되는 구간 중 길이가 가장 짧은 구간을 구하는 문제.

## 📌 풀이 방법
1. left와 right 포인터를 사용해 구간의 시작과 끝을 관리한다. 
2. 현재 합이 k보다 작거나 같으면 right를 이동하며 값을 더한다. 
3. 현재 합이 k보다 크면 left를 이동하며 값을 뺀다. 
4. 현재 합이 k와 같으면 구간 길이를 계산한다. 
5. 기존 구간보다 길이가 짧으면 시작 인덱스와 끝 인덱스를 갱신한다. 
6. 탐색이 끝나면 저장된 시작 인덱스와 끝 인덱스를 반환한다.

## ✅ 느낀 점
정렬된 양수 배열이라 투 포인터로 합을 늘리고 줄이는 흐름을 잡을 수 있었다.
