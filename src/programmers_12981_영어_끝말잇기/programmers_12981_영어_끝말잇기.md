## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 12981](https://school.programmers.co.kr/learn/courses/30/lessons/12981)
- 문제 요약: n명의 사람이 순서대로 단어를 이어 말하는 끝말잇기에서, 규칙(끝 글자 연결, 중복 금지, 한 글자 금지)을 어긴 첫 번째 탈락자의 번호와 차례를 구하는 문제.

## 📌 풀이 방법
1. 중복 단어 체크를 위해 Set을 사용하고 첫 번째 단어를 미리 저장한다.
2. 두 번째 단어부터 순회하면서 이전 단어와 현재 단어를 비교한다.
3. 이전 단어의 마지막 문자와 현재 단어의 첫 문자가 같은지 검사한다.
4. 끝말잇기 규칙을 만족하지 않거나 이미 등장한 단어이면 탈락 처리한다.
5. 탈락한 경우 (i % n) + 1로 사람 번호, (i / n) + 1로 차례를 계산한다.
6. 모든 단어를 통과하면 [0, 0]을 반환한다.

## ✅ 느낀 점
구현 문제는 조건을 정확히 나누는 것이 중요하다고 느꼈다.
