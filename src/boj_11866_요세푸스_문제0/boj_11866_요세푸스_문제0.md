## 💡 백준 문제
- 문제 링크: [백준 11866](https://www.acmicpc.net/problem/11866)
- 문제 요약: 원형으로 앉은 N명 중에서 K번째 사람을 반복적으로 제거하며 제거 순서를 출력하는 시뮬레이션 문제


## 📌 풀이 방법
1. `Deque`를 사용해 원형 큐를 구현했다.
2. K - 1번 동안 `pollFirst()`한 값을 `addLast()`하여 큐를 회전시켰다.
3. 이후 `pollFirst()`로 K번째 사람을 제거하며 순서를 기록했다.

## ✅ 느낀 점
addLast와 pollFirst를 사용하면 원형 구조를 직관적으로 표현할 수 있다는 점을 다시 한 번 정리할 수 있었다.
