## 💡 프로그래머  문제
- 문제 링크: [프로그래머스 12900](https://school.programmers.co.kr/learn/courses/30/lessons/12900)
- 문제 요약: 2 x n 크기의 바닥을 2 x 1 타일로 채우는 경우의 수를 구하는 문제.

## 📌 풀이 방법
1. n이 1일 때는 경우의 수가 1개이므로 dp[1] = 1로 초기화한다. 
2. n이 2일 때는 경우의 수가 2개이므로 dp[2] = 2로 초기화한다. 
3. n번째 경우의 수는 n-1번째에 세로 타일을 붙이는 경우와 n-2번째에 가로 타일 2개를 붙이는 경우의 합으로 계산한다. 
4. 각 값을 1,000,000,007로 나눈 나머지로 저장한다. 
5. dp[n]을 정답으로 반환한다.

## ✅ 느낀 점
타일을 채우는 경우의 수가 이전 두 경우의 합으로 이어지는 DP 구조라는 점이 핵심이었다.
