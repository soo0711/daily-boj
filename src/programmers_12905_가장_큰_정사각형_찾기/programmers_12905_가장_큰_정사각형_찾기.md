## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 12905](https://school.programmers.co.kr/learn/courses/30/lessons/12905)
- 문제 요약: 1로만 이루어진 가장 큰 정사각형의 넓이를 구하는 문제.

## 📌 풀이 방법
1. 완전탐색으로 정사각형을 찾으려 했지만, 모든 경우를 검사하면 시간초과가 발생함 
2. 각 위치를 기준으로 만들 수 있는 “정사각형의 한 변의 길이”를 저장하는 DP 방식으로 접근
3. (i, j)가 1일 경우, 왼쪽 / 위쪽 / 왼쪽 위 대각선 값 중 최솟값 + 1로 갱신 
4. 해당 값은 그 위치를 오른쪽 아래 꼭짓점으로 하는 최대 정사각형의 한 변 길이를 의미 반복하면서 가장 큰 값을 answer에 저장 
5. 최종적으로 answer * answer을 반환하여 넓이를 계산

## ✅ 느낀 점
처음에는 정사각형을 직접 찾으려고 해서 비효율적인 풀이를 떠올렸지만, 작은 정사각형의 정보를 활용해 큰 정사각형을 만드는 DP 아이디어가 중요했다. 
문제를 풀 때는 단순 구현보다 시간복잡도를 먼저 고려하는 습관이 중요하다고 느꼈다.
