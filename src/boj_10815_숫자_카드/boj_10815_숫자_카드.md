## 💡 백준 문제
- 문제 링크: [백준 10815](https://www.acmicpc.net/problem/10815)
- 문제 요약: 상근이가 가진 N개의 정수 카드 중에서, M개의 질문 숫자가 각각 존재하는지 여부를 1 또는 0으로 출력하는 문제

## 📌 풀이 방법
1. Set으로 상근이 가진 N개의 정수 카드를 받음.
2. Array로 M개의 질문 숫자 카드를 받음.
3. set.contains(array[i])로 set안에 배열의 값이 존재하는지 확인하고
4. 있으면 1, 없으면 0을 출력했다.

## ✅ 느낀 점
처음에는 array와 list로 접근해서 list.contains(array[i])를 사용했는데 list는 배열처럼 처음부터 끝까지 비교를 한다.
이걸 n번 반복하니까 O(n x m)의 복잡도가 나와서 시간초과가 됐다.

<br>
HashSet은 hashcode()를 계산해서 해당 위치만 확인한다. 그래서 이때는 시간 복잡도가 O(1) × M = O(M)가 나온다.

<br>

이 문제를 바탕으로 자료구조를 쓸 때는 시간복잡도도 계산해서 알맞은 자료구조를 선택해야겠다고 느꼈다.
