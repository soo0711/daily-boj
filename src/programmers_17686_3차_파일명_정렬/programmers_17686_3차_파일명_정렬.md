## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 17686](https://school.programmers.co.kr/learn/courses/30/lessons/17686)
- 문제 요약: 파일명을 HEAD, NUMBER 기준 우선순위에 따라 정렬하는 문제.

## 📌 풀이 방법
1. 파일명을 HEAD, NUMBER, 입력 순서로 분리해 저장하는 클래스를 만든다.
2. 파일명을 순회하며 숫자가 나오기 전까지는 HEAD, 이후 숫자 부분은 NUMBER로 추출한다. 
3. 비교 시 대소문자를 구분하지 않도록 HEAD를 소문자로 변환해 저장한다. 
4. NUMBER는 정수로 변환해 숫자 크기 기준으로 비교한다. 
5. HEAD가 같으면 NUMBER를 비교하고, 둘 다 같으면 입력 순서를 유지한다. 
6. 정렬 후 원래 파일명 순서대로 배열에 담아 반환한다.

## ✅ 느낀 점
문자열 전체를 비교하는 것이 아니라 HEAD와 NUMBER를 분리해 각각의 우선순위 기준으로 정렬하는 것이 핵심이었다.
