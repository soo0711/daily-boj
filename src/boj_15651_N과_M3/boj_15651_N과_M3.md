## 💡 백준 문제
- 문제 링크: [백준 15651](https://www.acmicpc.net/problem/15651)
- 문제 요약: 1부터 N까지의 자연수 중 중복을 허용해 M개를 선택한 모든 수열을 사전순으로 출력하는 문제

## 📌 풀이 방법
1. 백트래킹을 사용해 각 자리에 올 수 있는 숫자를 하나씩 선택했다.
2. `depth`를 통해 현재 몇 개의 숫자를 선택했는지를 관리했다.
3. 같은 숫자를 여러 번 선택할 수 있기 때문에 방문 여부를 확인하는 로직은 사용하지 않았다.
4. `depth == M`이 되면 하나의 수열이 완성되므로 결과를 출력했다.

## ✅ 느낀 점
중복 선택이 허용되는 조건 덕분에 이전 문제보다 로직이 단순해졌고 백트래킹의 기본 구조를 다시 한 번 정리할 수 있었다.
