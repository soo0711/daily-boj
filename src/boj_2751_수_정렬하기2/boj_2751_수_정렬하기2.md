## 💡 백준 문제
- 문제 링크: [백준 2751](https://www.acmicpc.net/problem/2751)
- 문제 요약: 주어진 N개의 정수를 오름차순으로 정렬하여 출력하는 문제


## 📌 풀이 방법
1. Arrays.sort를 사용해서 오름차순으로 정렬

## ✅ 느낀 점
1. System.out.println만 사용했을 때 <br>
→ 메모리 179,084KB / 실행 시간 4,992ms

2. StringBuilder로 출력을 모아서 처리했을 때  <br>
→ 메모리 144,980KB / 실행 시간 1,628ms

3. BufferedWriter를 사용했을 때  <br>
→ 메모리 122,896KB / 실행 시간 1,340ms

출력 방식을 개선할수록 메모리 사용량과 실행 시간이 모두 감소했다. <br>
특히 BufferedWriter가 가장 효율적인 결과를 보여주었다. <br>
대용량 출력 문제에서는 BufferedWriter 사용이 필수적이라는 것을 체감했다.
