## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 17677](https://school.programmers.co.kr/learn/courses/30/lessons/17677)
- 문제 요약: 두 문자열을 두 글자씩 끊어 다중집합을 만든 뒤 자카드 유사도를 구하고, 해당 값에 65536을 곱한 정수를 반환하는 문제.

## 📌 풀이 방법
1. 문자열을 소문자로 변환한 뒤 두 글자씩 끊으면서 영문자 쌍만 추출해 Map에 개수를 저장한다. 
2. 두 Map이 모두 비어있으면 65536을 반환한다. 
3. 같은 key에 대해 교집합은 min, 합집합은 max로 계산한다. 
4. 한쪽에만 존재하는 key는 합집합에만 더한다.
5. (교집합 / 합집합) * 65536 값을 구해 정수로 반환한다.

## ✅ 느낀 점
다중집합을 Map으로 표현하고 min/max로 교집합과 합집합을 계산하는 방식이 핵심이었다.
