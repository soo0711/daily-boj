## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 12985](https://school.programmers.co.kr/learn/courses/30/lessons/12985)
- 문제 요약: 토너먼트에서 번호가 재배정될 때, 참가자 A와 B가 몇 번째 라운드에서 서로 만나게 되는지 구하는 문제.

## 📌 풀이 방법
1. 참가자 번호는 매 라운드마다 다음 번호 = (현재 번호 + 1) / 2 규칙으로 갱신된다. 
2. A와 B가 같아지는 순간이 같은 경기에서 만나는 시점이므로, 두 값이 같아질 때까지 반복한다. 
3. 반복문에서 A와 B를 각각 (a / 2 + a % 2) 방식으로 갱신한다. (짝수는 a/2, 홀수는 a/2 + 1 효과)
4. 한 번 갱신될 때마다 라운드 수를 1씩 증가시킨다. 
5. A와 B가 같아지면 해당 시점의 카운트를 반환한다.
