## 💡 백준 문제
- 문제 링크: [백준 2485](https://www.acmicpc.net/problem/2485)
- 문제 요약: 이미 정렬된 가로수 위치가 주어질 때, 모든 가로수가 동일한 간격이 되도록 추가로 심어야 하는 가로수의 최소 개수를 구하는 문제


## 📌 풀이 방법
1. 인접한 가로수 사이의 거리들의 최대공약수를 구하기 위해 유클리드 호제법을 사용했다.
2. 첫 번째 간격을 기준으로 이후 모든 간격에 대해 누적 최대공약수를 계산했다.
3. 최종 간격을 기준으로 각 구간마다 추가로 심어야 하는 가로수의 개수를 계산해 더했다.
4. 한 구간에서 추가되는 가로수의 수는 `(구간 길이 / 간격) - 1` 이다.
5. 왜냐하면 양 끝의 기존 가로수를 제외하고 사이에 들어갈 가로수만 세기 때문이다.

## ✅ 느낀 점
유클리드 호제법을 이용해 여러 값의 최대공약수를 누적해서 구하는 방법을 이해할 수 있었다.
