## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 131127](https://school.programmers.co.kr/learn/courses/30/lessons/131127)
- 문제 요약: 10일 연속 할인 품목이 원하는 상품과 수량과 정확히 일치하는 시작 날짜의 개수를 구하는 문제.

## 📌 풀이 방법
1. 처음 10일 동안의 할인 품목을 Map에 담아 각 상품의 개수를 카운트한다. 
2. 현재 Map이 원하는 상품(want, number) 조건을 만족하는지 검사한다. 
3. 이후 슬라이딩 윈도우를 사용해 하루씩 이동하면서, 빠지는 상품은 -1, 새로 들어오는 상품은 +1로 Map을 갱신한다. 
4. 매 이동마다 조건을 만족하는지 확인하여 만족하면 answer를 증가시킨다.

## ✅ 느낀 점
슬라이딩 윈도우를 사용하여 매번 10일을 새로 계산하지 않고 문제를 해결할 수 있었다.
