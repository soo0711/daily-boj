## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 70129](https://school.programmers.co.kr/learn/courses/30/lessons/70129)
- 문제 요약: 문자열에서 0을 제거하고 남은 길이를 이진수로 변환하는 과정을 "1"이 될 때까지 반복하며
변환 횟수와 제거된 0의 총 개수를 구하는 문제.

## 📌 풀이 방법
1. 문자열이 "1"이 될 때까지 반복한다.
2. 문자열에서 0의 개수를 센다.
    - zero += 0의 개수
3. 0을 제거한 문자열의 길이를 구한다.
4. 해당 길이를 이진 문자열로 변환한다.
    - s = Integer.toBinaryString(length)
    - cnt++

## ✅ 느낀 점
처음에는 재귀로 직접 이진 변환을 구현하려고 했지만,
Java에서 제공하는 `Integer.toBinaryString()`을 활용하면
가독성과 성능 측면에서 더 효율적인 코드를 작성할 수 있다는 것을 알게 되었다.
