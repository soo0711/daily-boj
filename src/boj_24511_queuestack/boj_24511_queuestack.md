## 💡 백준 문제
- 문제 링크: [백준 24511](https://www.acmicpc.net/problem/24511)
- 문제 요약: 큐와 스택이 섞인 N개의 자료구조를 하나의 흐름으로 해석하여, 입력 값이 통과한 뒤 최종적으로 반환되는 값을 구하는 문제


## 📌 풀이 방법
1. 스택은 값을 넣자마자 다시 꺼내기 때문에 전체 흐름에서 결과에 영향을 주지 않는다고 판단했다.
2. 큐는 기존 값이 유지되며 뒤로 밀려나므로 큐에 해당하는 값들만 따로 관리하면 된다고 생각했다.
3. 큐에 들어 있는 값들만 `Deque`에 저장한 뒤, 입력 값이 들어올 때마다 앞에 추가하고 뒤에서 값을 꺼내는 방식으로 결과를 구했다.

## ✅ 느낀 점
- 모든 자료구조를 그대로 구현하기보다 실제로 결과에 영향을 주는 부분만 추려내는 것이 더 효율적인 해결 방법이라는 것을 배웠다.
