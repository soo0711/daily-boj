## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 43162](https://school.programmers.co.kr/learn/courses/30/lessons/43162)
- 문제 요약: 컴퓨터들의 연결 정보를 기반으로 서로 직·간접적으로 연결된 컴퓨터 그룹(네트워크)의 개수를 구하는 문제.

## 📌 풀이 방법
1. 각 컴퓨터의 방문 여부를 확인하기 위해 `visited` 배열을 생성한다.
2. 모든 컴퓨터를 순회하면서 아직 방문하지 않은 컴퓨터를 발견하면 새로운 네트워크가 시작된 것이므로 `answer`를 증가시킨다.
3. 방문하지 않은 컴퓨터를 시작점으로 `DFS`를 수행하여 연결된 모든 컴퓨터를 탐색한다.
4. 한 번의 DFS가 끝나면 하나의 네트워크에 속한 모든 컴퓨터가 방문 처리된다.


## ✅ 느낀 점
`DFS`를 사용하면 한 컴퓨터에서 시작해 연결된 모든 컴퓨터를 한 번에 방문 처리할 수 있어 네트워크 단위로 탐색할 수 있었다. 
`visited` 배열을 통해 이미 확인한 컴퓨터를 다시 탐색하지 않도록 하는 것이 중요하다는 점을 다시 한번 느꼈다.
