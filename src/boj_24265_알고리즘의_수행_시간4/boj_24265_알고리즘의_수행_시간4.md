## 💡 백준 문제
- 문제 링크: [백준 24265](https://www.acmicpc.net/problem/24265)
- 문제 요약: 입력 크기 n이 주어질 때 MenOfPassion 알고리즘에서 코드1`(sum <- sum + A[i] × A[j])`의 수행 횟수와 최고차항 차수를 출력해라.

## 📌 풀이 방법
```java
MenOfPassion(A[], n) {
        sum <- 0;
        for i <- 1 to n
            for j <- 1 to n
                sum <- sum + A[i] × A[j]; # 코드1
        return sum;
        }
```
1. n이 7일때 수행횟수 규칙 찾기
   - i = 1, 6번
   - i = 2, 5번
   - i = 3, 4번
   - i = 4, 3번
   - i = 5, 2번
   - i = 6, 1번
2. 1 + 2 + 3 + 4 + 5 + 6 = 21
3. 규칙 1 + 2 + 3 ... + (n-1)이 수행횟수 => n(n-1) / 2

## ✅ 느낀 점
이중 for문 규칙에 따라서 n(n-1)/2가 수행횟수이다. <br>
시간복잡도는 O(n^2)
