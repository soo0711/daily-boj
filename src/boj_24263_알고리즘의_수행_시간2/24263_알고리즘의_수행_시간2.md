## 💡 백준 문제
- 문제 링크: [백준 24263](https://www.acmicpc.net/problem/24263)
- 문제 요약: 입력 크기 n이 주어질 때 MenOfPassion 알고리즘에서 코드1`(sum += A[i])`의 수행 횟수와 최고차항 차수를 출력해라.
- 
## 📌 풀이 방법
```java
MenOfPassion(A[], n) {
    sum <- 0;
    for i <- 1 to n
        sum <- sum + A[i]; # 코드1
    return sum;
}
```
1. for문이 n번 반복되기 때문에 수행횟수는 n
2. 수행횟수는 n^1이므로 최고차수는 1

## ✅ 느낀 점
for문 반복 횟수는 입력 크기에 비례한다. <br>
시간복잡도는 O(n) = n^1
