## 💡 백준 문제
- 문제 링크: [백준 17103](https://www.acmicpc.net/problem/17103)
- 문제 요약: 짝수 N(≤1,000,000)을 두 소수의 합으로 나타내는 서로 다른 경우의 수를 구하는 문제

## 📌 풀이 방법
1. 에라토스테네스의 체를 이용해 1,000,000 이하의 소수 여부를 미리 구했다.
2. 각 N에 대해 2부터 N/2까지 반복하여 j와 N - j가 모두 소수인 경우를 세어 골드바흐 파티션의 개수를 구했다.

## ✅ 느낀 점
에라토스테네스의 체를 이용해 소수를 미리 구해두면 매번 소수 판별을 할 필요가 없어지고
j + (N - j) 구조를 사용해 중첩 반복문 없이 문제를 해결할 수 있다는 점을 깨달았다.
