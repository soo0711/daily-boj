## 💡 백준 문제
- 문제 링크: [백준 24313](https://www.acmicpc.net/problem/24313)
- 문제 요약: 함수 f(n) = a₁n + a₀가 주어졌을 때, 주어진 c와 n₀에 대해 f(n)이 O(n) 정의를 만족하는지 판별해라.

## 📌 풀이 방법
```test
O(g(n)) = {f(n) | 모든 n ≥ n0에 대하여 f(n) ≤ c × g(n)인 양의 상수 c와 n0가 존재한다}
```
1. f(n) <= c * g(n)
2. an * n <= c * n
3. (an - c) * n <= 0
4. 으로 이 공식이 성립하려면 n은 양수이기 때문에 an - c <= 0 이여야한다.
5. an <= c 이고 an * n <= c * n 이 공식이 성립하면
6. O(n) 정의를 만족한다.

## ✅ 느낀 점
처음에는 단순히 n₀에서만 부등식을 확인했었는데 모든 n ≥ n₀에서 항상 성립해야 한다는 점 조건이 있었다. 이 조건을 적용해서 다시 문제를 풀면서 정의를 이해하게 되었다. <br>
그리고 빅오를 단순히 '시간 복잡도가 n에 비례한다' 정도로만 이해했는데 이 문제를 통해서 정의가 '모든 n ≥ n₀에 대해 f(n) ≤ c × g(n)'임을 알게 됐다.
