## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 87390](https://school.programmers.co.kr/learn/courses/30/lessons/87390)
- 문제 요약: n×n 배열을 특정 규칙으로 채운 뒤 1차원으로 펼쳤을 때, left부터 right까지의 구간만 잘라 반환하는 문제.

## 📌 풀이 방법
1. 전체 2차원 배열을 만들지 않고, 1차원 인덱스를 기준으로 직접 값 계산 
2. 각 인덱스 i에 대해 행 = i / n, 열 = i % n으로 변환 
3. 해당 위치 값은 (행+1, 열+1) 중 큰 값이라는 규칙을 이용 
4. left부터 right까지 순회하며 위 규칙으로 바로 값 채우기

## ✅ 느낀 점
배열을 직접 만들지 않고 규칙으로 접근하는 것이 중요하다는 것을 느꼈다.
