## 💡 백준 문제
- 문제 링크: [백준 10989](https://www.acmicpc.net/problem/10989)
- 문제 요약: 최대 1천만 개의 자연수를 주어 메모리 제한 8MB 내에서 오름차순으로 정렬해 출력하는 문제

## 📌 풀이 방법
1. index를 사용해서 index에 값이 있으면 출력 함.

## ✅ 느낀 점
Arrays.sort, Buffered를 사용해서 풀었을 때 <br>
-> 메모리 343448kb, 시간 2468ms

idx를 사용했을 때 <br>
-> 메모리 300680kb, 시간 1692ms <br>

idx를 사용했을 때 시간이 거의 반이나 줄어들었다. <br>
정렬은 Arrays.sort로 해야하다는 고정관념이 있었는데 idx를 사용함으로써 새로운 정렬 방법을 알았다.
