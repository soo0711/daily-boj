## 💡 백준 문제
- 문제 링크: [백준 15649](https://www.acmicpc.net/problem/15649)
- 문제 요약: 중복 없이 M개를 선택해 만들 수 있는 모든 순열을 사전순으로 출력하는 문제

## 📌 풀이 방법
1. 백트래킹을 사용해 각 자리에 올 수 있는 숫자를 하나씩 선택했다.
2. `depth`를 통해 현재 몇 개의 숫자를 선택했는지를 관리했다.
3. 이미 사용한 숫자는 다시 선택하지 않기 위해 `visited` 배열로 사용 여부를 체크했다.
4. 숫자를 하나 선택하면 다음 자리로 재귀 호출하고 탐색이 끝나면 방문 상태를 되돌려 다른 경우를 탐색했다.
5. `depth == M`이 되면 하나의 수열이 완성되므로 결과를 출력했다.

## ✅ 느낀 점
백트래킹은 `선택 → 탐색 → 되돌리기`의 흐름을 코드로 표현하는 방식임을 알게 되었다.
`depth`, `visited`, `결과` 배열의 역할을 명확히 구분하니 문제 구조가 훨씬 잘 보였다.
