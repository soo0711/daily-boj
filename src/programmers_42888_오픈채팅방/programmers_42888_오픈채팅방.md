## 💡 백준 문제
- 문제 링크: [프로그래머스 42888](https://school.programmers.co.kr/learn/courses/30/lessons/42888)
- 문제 요약: 유저 ID 기반 기록을 추적해 입장/퇴장 메시지를 남기고, 닉네임 변경 사항을 모두 반영한 최종 채팅 로그를 출력하는 문제

## 📌 풀이 방법
1. 유저 ID 기준으로 현재 닉네임을 관리한다.
2. 전체 기록을 한 번 순회하며 닉네임 상태를 갱신하고 출력 개수를 계산했다.
3. 다시 한 번 기록을 순회하며 최종 메시지를 생성했다.


## ✅ 느낀 점
- 닉네임이 아니라 유저 ID가 핵심 키라는 점을 이해하는 게 가장 중요했다.
