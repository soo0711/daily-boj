## 💡 백준 문제
- 문제 링크: [백준 1929](https://www.acmicpc.net/problem/1929)
- 문제 요약: 주어진 범위 [M,N] 안에 포함된 모든 소수를 오름차순으로 출력하는 문제


## 📌 풀이 방법
1. 에라토스테네스의 체를 사용해서 문제를 풀어줬다.
2. 0과 1은 소수가 아니므로 boolean 배열에서 true로 표시했다.
3. 2부터 √N까지 반복하며 소수인 경우 해당 수의 제곱부터 배수를 제거해줬다.
4. boolean 배열에서 false로 남아 있는 수만 소수이므로 이를 출력했다.

## ✅ 느낀 점
에라토스테네스의 체를 사용하면 구간 내 소수를 출력하는 문제를 효율적으로 해결할 수 있었다.  <br>
이번 문제를 통해서 에라토스테네스의 체를 완벽히 이해할 수 있었다.
