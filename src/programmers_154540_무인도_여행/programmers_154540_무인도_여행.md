## 💡 백준 문제
- 문제 링크: [프로그래머스 154540](https://school.programmers.co.kr/learn/courses/30/lessons/154540)
- 문제 요약: 격자 지도에서 상하좌우로 연결된 숫자 영역(섬)마다 값을 합산해 생존 가능 일수를 구하고, 이를 오름차순으로 반환하는 문제 (섬이 없으면 -1 반환).

## 📌 풀이 방법
1. 문자열 배열 maps를 2차원 배열로 변환한다.
2. 전체 좌표를 순회하면서 "X"가 아니고 방문하지 않은 칸이면 BFS를 시작한다.
3. BFS를 통해 상하좌우로 연결된 모든 칸을 탐색하며 값을 누적한다.
4. 탐색한 칸은 "0"으로 변경하여 재방문을 방지한다.
5. 하나의 BFS가 끝나면 해당 섬의 총합을 리스트에 저장한다.
6. 모든 탐색이 끝난 후 리스트를 오름차순으로 정렬한다.
7. 리스트가 비어있으면 [-1]을 반환하고, 아니면 배열로 변환하여 반환한다.


## ✅ 느낀 점
연결된 영역을 탐색하는 BFS 패턴을 정확히 이해하고 적용하는 것이 핵심이었다.
