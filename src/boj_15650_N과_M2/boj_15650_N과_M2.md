## 💡 백준 문제
- 문제 링크: [백준 15650](https://www.acmicpc.net/problem/15650)
- 문제 요약: 중복 없이 M개를 선택해 오름차순 조합을 출력하는 문제

## 📌 풀이 방법
1. 백트래킹을 사용해 각 자리에 올 수 있는 숫자를 하나씩 선택했다.
2. `depth`를 통해 현재 몇 개의 숫자를 선택했는지를 관리했다.
3. 오름차순 조합을 유지하기 위해 `start` 값을 함께 전달하여 이전에 선택한 숫자보다 작은 값은 제외했다.
4. 이미 사용한 숫자는 다시 선택하지 않기 위해 `visited` 배열로 사용 여부를 체크했다.
5. 숫자를 하나 선택하면 다음 자리로 재귀 호출하고 탐색이 끝나면 방문 상태를 되돌려 다른 경우를 탐색했다.
6. `depth == M`이 되면 하나의 수열이 완성되므로 결과를 출력했다.

## ✅ 느낀 점
- `start`를 활용해 오름차순 조건을 깔끔하게 처리할 수 있었다.
