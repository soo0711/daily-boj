## 💡 백준 문제
- 문제 링크: [백준 4134](https://www.acmicpc.net/problem/4134)
- 문제 요약: 주어진 정수 n 이상에서 가장 작은 소수를 찾아 출력하는 문제


## 📌 풀이 방법
1. 입력으로 받은 n부터 시작하여 소수인지 판별하고 소수가 아니면 n을 1씩 증가시키며 다시 검사한다.
2. 어떤 수 n이 소수인지 판별하기 위해 `2 <= i <= sqrt(n)` 범위의 정수로 나누어 보며, 나누어 떨어지면 합성수이다.
3. 0과 1은 소수가 아니므로 예외 처리했다.
4. 위 조건을 만족하는 가장 작은 소수를 찾으면 출력했다.

## ✅ 느낀 점
이 문제는 정수 n의 범위가 `0 ≤ n ≤ 4 × 10⁹`이므로 `long` 자료형을 사용해야 한다.  
소수 판별 시 `sqrt(n)`까지만 확인해도 충분하다는 점을 정확히 이해할 수 있었다.
