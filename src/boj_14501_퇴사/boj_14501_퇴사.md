## 💡 백준 문제
- 문제 링크: [백준 14501](https://www.acmicpc.net/problem/14501)
- 문제 요약: 주어진 상담 일정에서 각 상담의 기간과 보상을 고려하여, 퇴사일(N+1일) 이전까지 겹치지 않게 상담을 선택했을 때 얻을 수 있는 최대 수익을 구하는 문제


## 📌 풀이 방법
1. i일 아침에 도착했을 때의 최대 수익을 생각했다.
2. 선택지로는 상담을 한다 / 상담을 안 한다가 있었다.
3. 상담을 하지 않으면 → `dp[i + 1] = max(dp[i + 1], dp[i])`
4. 상담을 하면 → `dp[i + T[i]] = max(dp[i + T[i]], dp[i] + P[i])`
5. n+1일에 퇴사하므로, `dp[n + 1]`을 출력했다.

## ✅ 느낀 점
처음엔 재귀로 모든 경우를 탐색하려 했지만 미래로 최대값을 전달하는 방식을 활용하니 시간 복잡도와 공간 복잡도가 O(n)으로 줄었다.
