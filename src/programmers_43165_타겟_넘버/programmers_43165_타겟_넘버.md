## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 43165](https://school.programmers.co.kr/learn/courses/30/lessons/43165)
- 문제 요약: 주어진 숫자 배열에서 각 숫자에 + 또는 -를 붙여 합이 target이 되는 모든 경우의 수를 구하는 문제.

## 📌 풀이 방법
1. 입력 예시를 보면 다음과 같이 각 숫자마다 + 또는 - 두 가지 선택이 존재한다
```
0
-4 +4
-1 + 1 -1 + 1
-2 +2 -2 +2 -2 +2 -2 +2
.
.
.
```
2. 이런 구조는 모든 경우의 수를 탐색하는 DFS(완전 탐색) 형태라고 판단했다.
3. 각 숫자에 대해 + 또는 - 두 가지 경우를 재귀적으로 탐색한다.
4. depth로 현재 사용할 숫자의 위치를 관리하고 sum에 값을 누적한다.
5. 모든 숫자를 사용했을 때 (depth == numbers.length) sum == target이면 경우의 수를 증가시킨다.

## ✅ 느낀 점
각 숫자마다 두 가지 선택이 있기 때문에 DFS를 이용한 완전탐색 문제였다. 재귀로 +와 - 두 가지 분기를 만들면서 탐색하는 방식에 익숙해질 수 있었다.
