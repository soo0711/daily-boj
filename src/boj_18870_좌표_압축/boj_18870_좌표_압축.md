## 💡 백준 문제
- 문제 링크: [백준 18870](https://www.acmicpc.net/problem/18870)
- 문제 요약: 서로 다른 좌표들을 정렬해 순위를 매기고, 각 원래 좌표를 그 순위로 바꿔 출력하는 문제

## 📌 풀이 방법
1. 배열 2개 받아서 배열 1개를 정렬해준다.
2. Map을 사용해서 key값을 넣으면 정렬값이 나올 수 있게 Map을 생성해서 key, value값을 넣는다.
3. 이때 key값이 없을 때만 Map에 put해준다.
4. 원래 배열을 이용해서 key값을 입력하면 value가 출력되게 했다.

## ✅ 느낀 점
Map의 key, value값을 활용하니 이중 for문을 쓰지 않고 풀 수 있었다. <br>
원본 배열과 정렬 배열을 두 개 이용해야지 원본 좌표를 정렬 순위대로 출력할 수 있었다.
