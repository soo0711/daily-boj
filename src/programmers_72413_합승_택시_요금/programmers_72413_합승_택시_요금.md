## 💡 백준 문제
- 문제 링크: [프로그래머스 72413](https://school.programmers.co.kr/learn/courses/30/lessons/72413)
- 문제 요약: 출발점 s에서 시작해 특정 지점 k까지는 같이 이동하고, 이후 각자 a와 b로 갈 때의 최소 총 택시요금을 구하는 문제.

## 📌 풀이 방법
1. fares를 이용해 각 지점 간 이동 비용을 저장한 양방향 그래프를 생성한다.
2. 출발지 s, 도착지 a, 도착지 b 각각에서 다익스트라를 수행해 모든 지점까지의 최단거리를 구한다.
3. 모든 지점 i를 합승 지점으로 가정하고 s → i + i → a + i → b 비용을 계산한다.
4. 위 값들 중 최솟값을 선택하여 최종 답으로 반환한다.


## ✅ 느낀 점
처음에는 BFS로 접근하려 했지만, 가중치가 있는 그래프라 다익스트라가 필요함을 깨달았다.
s, a, b 각각에서 최단거리 배열을 구해 조합하는 방식이 효율적이라는 걸 배웠다.
