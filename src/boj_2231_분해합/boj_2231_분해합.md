## 💡 백준 문제
- 문제 링크: [백준 2231](https://www.acmicpc.net/problem/2231)
- 문제 요약: 자연수 N이 주어졌을 때, 분해합이 N이 되는 가장 작은 수를 찾는 문제


## 📌 풀이 방법
1. n = 256에서부터 for문을 돌려 i = n 일 때 -1씩 빼서 모든 경우의 수를 시도해보려고 했다.
2. 각 자리 수를 더하기 위해서 abc는 분배합으로
3. abc += n % 10 -> n /= 10 공식을 써서 각 자리의 수를 더했다.
4. sum = abc + i로 n == sum일 때 min > i이면 min은 가장 작은 생성자로 바꿔주었다.

## ✅ 느낀 점
각 자리 수를 어떻게 더하면 좋을지 고민을 많이 했던 것 같다. 처음에는 맨 앞자리부터 잘라야한다고 생각했는데 다 더할 것이기 때문에 자리 수는 상관없었다.
<br>
각 자리 수를 뽑는 상황에서 n % 10, n /= 10 공식을 사용할 수 있으니 기억하고 있어야겠다.
