## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 86971](https://school.programmers.co.kr/learn/courses/30/lessons/86971)
- 문제 요약: 전선 하나를 끊었을 때 두 전력망의 송전탑 개수 차이가 최소가 되는 값을 구하는 문제.

## 📌 풀이 방법
1. wires를 하나씩 끊어보며 경우를 확인한다. 
2. 끊은 전선은 DFS 탐색에서 제외한다. 
3. 끊어진 한쪽 송전탑에서 DFS를 시작해 연결된 송전탑 개수를 구한다. 
4. 다른 쪽 전력망의 송전탑 개수는 전체 n에서 탐색한 개수를 뺀 값으로 구한다. 
5. 두 전력망의 송전탑 개수 차이를 계산한다. 
6. 모든 전선을 확인하며 가장 작은 차이를 반환한다.

## ✅ 느낀 점
전선 개수가 많지 않아 하나씩 끊어보면서 DFS로 연결 개수를 세는 방식으로 풀 수 있었다.
