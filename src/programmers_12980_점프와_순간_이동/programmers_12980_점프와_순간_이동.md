## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 12980](https://school.programmers.co.kr/learn/courses/30/lessons/12980)
- 문제 요약: 순간이동은 비용이 없고 점프만 비용이 들 때, 거리 N까지 이동하기 위한 최소 점프 횟수(건전지 사용량)를 구하는 문제.

## 📌 풀이 방법
1. 목표 지점 N에서 시작하여 1이 될 때까지 역으로 탐색한다.
2. N이 짝수라면 순간이동으로 온 것이므로 건전지 사용 없이 N / 2로 이동한다.
3. N이 홀수라면 반드시 점프를 한 번 사용해야 하므로 건전지 사용량을 나머지만큼 증가시키고 N - 1 후 N / 2로 이동한다.
4. 이를 반복하면서 홀수일 때의 횟수를 누적하면 최소 점프 횟수가 된다.
5. 즉, N을 2로 나누는 과정에서 발생하는 나머지의 개수를 세는 것과 동일하다.

## ✅ 느낀 점
문제를 단순 구현처럼 접근하면 경우의 수가 많아 보여 복잡하게 느껴졌지만, 규칙을 찾아보니 매우 간단한 수학적 패턴으로 해결할 수 있었다.
