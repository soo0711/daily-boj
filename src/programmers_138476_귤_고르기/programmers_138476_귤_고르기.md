## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 138476](https://school.programmers.co.kr/learn/courses/30/lessons/138476)
- 문제 요약: 귤 k개를 선택할 때, 크기 종류 수가 최소가 되도록 가장 많이 나온 크기부터 선택해 필요한 종류 개수를 구하는 문제.

## 📌 풀이 방법
1. 귤 크기별 개수를 세기 위해 HashMap에 빈도를 저장한다. 
2. 각 크기의 개수를 기준으로 내림차순 정렬한다. 
3. 개수가 많은 크기부터 차례대로 선택하면서 k개를 채운다. 
4. k개 이상이 되는 순간까지 사용한 크기의 개수를 반환한다.


## ✅ 느낀 점
많이 등장하는 값을 우선 선택하면 종류 수를 최소화할 수 있었다.
