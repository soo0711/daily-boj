## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 155651](https://school.programmers.co.kr/learn/courses/30/lessons/155651)
- 문제 요약: 예약 시간과 청소 시간을 고려해 필요한 최소 객실 수를 구하는 문제.

## 📌 풀이 방법
1. 예약 시간을 분 단위로 변환한다. 
2. 퇴실 시간에는 청소 시간 10분을 더해 저장한다. 
3. 예약 목록을 입실 시간 기준으로 정렬한다. 
4. 우선순위 큐에 현재 사용 중인 객실의 사용 가능 시간을 저장한다. 
5. 가장 빠른 사용 가능 시간이 현재 입실 시간보다 작거나 같으면 해당 객실을 재사용한다. 
6. 모든 예약을 처리한 뒤 우선순위 큐의 크기를 반환한다.

## ✅ 느낀 점
퇴실 시간에 청소 시간까지 더해서 객실이 다시 사용 가능한 시점을 기준으로 관리하면 쉽게 풀 수 있었다.
