## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 64065](https://school.programmers.co.kr/learn/courses/30/lessons/64065)
- 문제 요약: 문자열로 표현된 여러 집합 정보를 바탕으로, 원소의 등장 순서를 이용해 원래의 튜플을 복원하는 문제.

## 📌 풀이 방법
1. 문자열을 순회하면서 {} 단위로 집합을 분리한다.
2. 각 집합을 문자열 형태로 리스트에 저장한다.
3. 집합의 길이를 기준으로 오름차순 정렬한다. → 원소 개수가 적은 집합부터 확인하기 위함
4. 순서대로 순회하면서 Set을 이용해 이미 등장한 숫자인지 체크한다.
5. 처음 등장하는 숫자만 결과 리스트에 추가한다.
6. 최종적으로 리스트를 배열로 변환하여 반환한다.

## ✅ 느낀 점
문자열 파싱과 자료구조(Set)의 특성을 잘 활용하면 복잡해 보이는 문제도 단순한 흐름으로 풀 수 있다는 걸 느꼈다.
