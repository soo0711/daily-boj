## 💡 백준 문제
- 문제 링크: [프로그래머스 131701](https://school.programmers.co.kr/learn/courses/30/lessons/131701)
- 문제 요약: 원형 수열에서 가능한 모든 연속 부분 수열의 합을 구했을 때, 서로 다른 합의 개수를 구하는 문제.
## 📌 풀이 방법
1. 중복을 제거하기 위해 Set을 사용한다.
2. 시작 인덱스를 0부터 n-1까지 순회한다.
3. 각 시작점에서 길이를 1부터 n까지 늘려가며 연속 부분 수열의 합을 구한다.
4. 원형 구조이므로 (i + j) % n을 사용해 인덱스를 순환시킨다.
5. 매번 구한 합을 Set에 넣어 중복을 제거한다.
6. 최종적으로 Set의 크기를 반환한다.

## ✅ 느낀 점
원형 수열이라 복잡하게 생각했지만 % 연산으로 간단하게 해결할 수 있었고, 중복 제거는 Set으로 깔끔하게 처리할 수 있었다.
