## 💡 백준 문제
- 문제 링크: [백준 19532](https://www.acmicpc.net/problem/19532)
- 문제 요약: 주어진 2×2 연립방정식에서 x와 y를 계산해, 각 값을 -999~999 사이의 정수로 제출하는 문제

## 📌 풀이 방법
1. 연립방정식의 x와 y를 찾는 공식을 활용해서 풀었다.
2. x = ce - bf / ae - bd
3. y = af - cd / ae - bd

## ✅ 느낀 점
-999 ~ 999를 다 대입해봐야하는 건 줄 알았는데 x와 y를 구하는 공식이 존재해서 쉽게 풀었다.
