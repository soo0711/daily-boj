## 💡 백준 문제
- 문제 링크: [백준 24262](https://www.acmicpc.net/problem/24262)
- 문제 요약: 입력 크기 n이 주어질 때 MenOfPassion 알고리즘에서 코드1의 수행 횟수와 최고차항 차수를 출력해라

## 📌 풀이 방법
```
MenOfPassion(A[], n) {
i = ⌊n / 2⌋;
return A[i];  // 코드1
}
```
1. A[i]는 항상 한 번만 실행 => 실행횟수는 1번
2. 1은 n^0 이므로 최고차항은 0이다.

## ✅ 느낀 점
상수 실행 횟수는 늘 1!
