## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 17683](https://school.programmers.co.kr/learn/courses/30/lessons/17683)
- 문제 요약: 기억한 멜로디와 재생 시간을 기준으로 실제 재생된 음악을 비교해, 조건에 맞는 곡 중 가장 오래 재생된 음악의 제목을 찾는 문제.

## 📌 풀이 방법
1. `#` -> 이 있는 건 다른 소문자로 바꿔준다. (왜냐면 1분에 한 음)
2. H * 60 + M 끝난 시각 - 시작 시간 => 숫자
3. 숫자만큼 음악의 음을 늘리거나 자른다.
4. m이 들어있는지 확인한다.
5. 일치하는 곡이 여러개면 재생된 시간이 긴지 비교해준다.

## ✅ 느낀 점
`#` 음 처리 때문에 문자열 비교 전에 전처리가 중요했다.
