## 💡 백준 문제
- 문제 링크: [백준 1269](https://www.acmicpc.net/problem/1269)
- 문제 요약: 두 자연수 집합 A와 B가 주어질 때, 서로 겹치지 않는 원소들만 모은 대칭 차집합의 원소 개수를 구하는 문제


## 📌 풀이 방법
1. 대칭 차집합의 개수는 `N + M - 2 * 교집합의 개수`다.
2. 그래서 setA, setB로 각 집합을 입력받고
3. 교집합을 저장할 set을 만들어서 `interaction.retainAll(setB)` 매서드를 사용했다.
4. `N + M - 2 * interaction.size()` 공식을 활용해서 대칭 차집합의 개수를 출력했다.

## ✅ 느낀 점
대칭차집합의 개수를 구하는 공식과 Set의 교집합, 차집합을 구하는 매서드를 활용해서 쉽게 풀 수 있었다.
