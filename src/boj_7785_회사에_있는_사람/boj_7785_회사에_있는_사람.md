## 💡 백준 문제
- 문제 링크: [백준 7785](https://www.acmicpc.net/problem/7785)
- 문제 요약: 출입 로그를 보고 현재 회사에 남아 있는 직원들을 찾아 이름을 사전순 역순으로 출력하는 문제

## 📌 풀이 방법
1. 출근한 사람은 삽입하고 퇴근한 사람은 제거해서 역순으로 정렬하는 문제라고 해석했다.
2. 삽입과 제거가 많을 때는 Set 자료구조를 활용해야지 시간 복잡도가 적다.
3. Set을 사용해 add, remove를 해주고 Set을 List로 변화해 역순으로 정렬하고 출력해줬다.

## ✅ 느낀 점
HashSet은 내부적으로 hashCode()를 이용해 저장 위치를 계산하기 때문에
삽입, 삭제, 조회 연산이 평균적으로 O(1)의 시간 복잡도를 가진다. <br>

따라서 삽입과 제거가 많이 발생하는 문제에서는 Set 자료구조가 적절하다고 느꼈다.
