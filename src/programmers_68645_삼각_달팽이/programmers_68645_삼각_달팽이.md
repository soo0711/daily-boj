## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 68645](https://school.programmers.co.kr/learn/courses/30/lessons/68645)
- 문제 요약: 삼각형 형태의 배열을 반시계 방향 달팽이 순서로 채운 뒤 결과를 1차원 배열로 반환하는 문제.

## 📌 풀이 방법
1. n x n 크기의 2차원 배열을 생성한다. 
2. 아래, 오른쪽, 대각선 위 방향을 반복하며 숫자를 채운다. 
3. i번째 반복에서는 n - i개의 칸을 이동하며 값을 저장한다. 
4. 방향은 아래 → 오른쪽 → 대각선 위 순으로 순환한다. 
5. 모든 숫자를 채운 뒤 삼각형 영역만 순서대로 꺼낸다. 
6. 1차원 배열에 담아 반환한다.

## ✅ 느낀 점
달팽이 이동 규칙을 찾고 방향 전환 패턴을 구현하는 것이 중요했던 문제였다.
