## 💡 백준 문제
- 문제 링크: [백준 4779](https://www.acmicpc.net/problem/4779)
- 문제 요약: 정수 N(0 ≤ N ≤ 12) 이 주어질 때, 길이 3ⁿ의 문자열에서 시작해 가운데 1/3을 반복적으로 공백으로 제거하여 만든 칸토어 집합의 최종 문자열 형태를 출력하는 문제.

## 📌 풀이 방법
1. 재귀 함수가 하나의 구간(start, length)을 책임지도록 정의했다.
2. 더 이상 쪼갤 수 없는 길이 1인 경우를 종료 조건으로 설정했다.
3. 현재 구간의 가운데 1/3을 공백으로 처리했다.
4. 앞과 뒤 1/3 구간은 재귀 호출로 다음 단계로 넘겼다.

## ✅ 느낀 점
재귀를 전체 흐름으로 보지 않고 구간 단위로 역할을 나눠서 생각하니 재귀 문제를 훨씬 수월하게 풀 수 있었다.
