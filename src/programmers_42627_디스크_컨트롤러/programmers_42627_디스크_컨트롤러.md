## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 42627](https://school.programmers.co.kr/learn/courses/30/lessons/42627)
- 문제 요약: 디스크 작업들을 우선순위에 따라 처리했을 때 작업 요청부터 종료까지 걸린 평균 시간을 구하는 문제.

## 📌 풀이 방법
1. jobs를 요청 시각 기준으로 오름차순 정렬한다.
2. 현재 시각까지 요청된 작업들을 우선순위 큐에 넣는다.
3. 큐에서는 소요 시간이 짧은 작업을 먼저 꺼내 처리한다.
4. 처리한 작업의 종료 시각에서 요청 시각을 뺀 값을 누적한다.
5. 큐가 비어 있으면 다음 작업 요청 시각으로 현재 시간을 이동한다.
6. 모든 작업을 처리한 뒤 누적 시간을 작업 개수로 나누어 반환한다.

## ✅ 느낀 점
현재 시각에 처리 가능한 작업만 큐에 넣고, 소요 시간이 짧은 작업부터 처리하는 흐름이 중요했다.
