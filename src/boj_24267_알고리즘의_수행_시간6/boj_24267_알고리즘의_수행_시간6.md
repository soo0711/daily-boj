## 💡 백준 문제
- 문제 링크: [백준 24266](https://www.acmicpc.net/problem/24266)
- 문제 요약: 입력 크기 n이 주어질 때 MenOfPassion 알고리즘에서 코드1`(sum <- sum + A[i] × A[j] × A[k])`의 수행 횟수와 최고차항 차수를 출력해라.

## 📌 풀이 방법
```java
MenOfPassion(A[], n) {
    sum <- 0;
    for i <- 1 to n - 2
        for j <- i + 1 to n - 1
            for k <- j + 1 to n
                sum <- sum + A[i] × A[j] × A[k]; # 코드1
    return sum;
}
```
1. n이 7일때 수행횟수 규칙 찾기
    - i=1 → j=2~6 → k=j+1~7 → 합 = 5+4+3+2+1 = 15 
    - i=2 → j=3~6 → k=j+1~7 → 합 = 4+3+2+1 = 10 
    - i=3 → j=4~6 → k=j+1~7 → 합 = 3+2+1 = 6 
    - i=4 → j=5~6 → k=j+1~7 → 합 = 2+1 = 3 
    - i=5 → j=6 → k=7 → 합 = 1 
    - i=6 → j=7 → k 없음 → 0
2. 15 + 10 + 6 + 3 + 1 = 35
3. 일반화 규칙 수행횟수 => n(n-1)(n-2) / 6

## ✅ 느낀 점
삼중 for문 규칙에 따라서 n(n-1)(n-2)/6이 수행횟수이다. <br>
시간복잡도는 O(n^3)
