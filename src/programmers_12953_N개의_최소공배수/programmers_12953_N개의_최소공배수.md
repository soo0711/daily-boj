## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 12953](https://school.programmers.co.kr/learn/courses/30/lessons/12953)
- 문제 요약: 여러 수가 모두 나누어떨어지는 가장 작은 공통 배수(최소공배수)를 구하는 문제.

## 📌 풀이 방법
1. 배열의 첫 번째 값을 기준으로 시작해 누적 최소공배수를 구한다.
2. 두 수의 최소공배수는 (a * b) / gcd(a, b) 공식을 이용한다.
3. gcd는 유클리드 호제법을 사용해 효율적으로 계산한다.
4. 배열을 순회하면서 이전까지의 최소공배수와 다음 값을 계속 갱신해 나간다.
5. 최종적으로 누적된 값을 반환한다.


## ✅ 느낀 점
유클리드 호제법을 활용하면 최대공약수를 빠르게 구할 수 있었다.
