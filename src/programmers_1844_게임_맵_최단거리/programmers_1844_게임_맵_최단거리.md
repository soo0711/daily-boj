## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 1844](https://school.programmers.co.kr/learn/courses/30/lessons/1844)
- 문제 요약: 벽이 있는 맵에서 시작점에서 목표점까지 이동 가능한 최단 경로 칸 수를 구하는 문제.

## 📌 풀이 방법
1. 맵에서 벽(0)을 피해 시작점(1,1)에서 목표점(n,m)까지 이동 가능한 최단 거리를 구함.
2. 상하좌우로 한 칸씩 이동하며 최단 거리를 찾기 위해 BFS 사용.
3. dist 배열로 각 칸까지 이동한 거리 기록, 큐에 시작점(0,0) 추가.
4. 큐에서 좌표를 꺼내 상하좌우 탐색, 이동 가능하고 아직 방문하지 않은 칸 거리 갱신 후 큐에 추가.
5. 목표점에 도달했으면 dist[n-1][m-1] 반환, 도달 불가 시 -1 반환.

## ✅ 느낀 점
BFS를 사용하면 최단 경로 문제를 직관적으로 풀 수 있어 알고리즘 선택이 중요함을 알게 되었다.
