## 💡 백준 문제
- 문제 링크: [백준 15652](https://www.acmicpc.net/problem/15652)
- 문제 요약: 자연수 1부터 N까지 중에서 중복을 허용해 M개를 선택하되 비내림차순을 유지하는 모든 수열을 사전순으로 출력하는 백트래킹 문제
- 
## 📌 풀이 방법
1. 백트래킹을 사용해 각 자리에 올 수 있는 숫자를 하나씩 선택했다.
2. `depth`를 통해 현재 몇 개의 숫자를 선택했는지를 관리했다.
3. 비내림차순을 유지하기 위해 이전에 선택한 값보다 작은 수는 선택하지 않도록 `start` 값을 사용한다.
4. 같은 숫자를 여러 번 선택할 수 있기 때문에 방문 여부를 확인하는 로직은 사용하지 않았다.
5. `depth == M`이 되면 하나의 수열이 완성되므로 결과를 출력했다.

## ✅ 느낀 점
백트래킹 문제에서 조건에 따라 상태를 줄이는 방식이 얼마나 중요한지 다시 한 번 느꼈다.
