## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 12907](https://school.programmers.co.kr/learn/courses/30/lessons/12907)
- 문제 요약: 주어진 화폐 단위로 n원을 만들 수 있는 경우의 수를 구하는 문제.

## 📌 풀이 방법
1. 경우의 수를 저장할 dp 배열을 만들고 dp[0]을 1로 초기화한다. 
2. 각 화폐 단위를 하나씩 기준으로 잡는다. 
3. 현재 화폐 단위부터 n원까지 순회한다. 
4. dp[i]에 dp[i - 현재 화폐] 값을 더해 현재 금액을 만드는 경우의 수를 누적한다. 
5. 모든 값은 1,000,000,007로 나눈 나머지로 저장한다. 
6. dp[n]을 정답으로 반환한다.

## ✅ 느낀 점
화폐를 하나씩 기준으로 누적해야 중복 조합을 피하면서 경우의 수를 계산할 수 있었다.
