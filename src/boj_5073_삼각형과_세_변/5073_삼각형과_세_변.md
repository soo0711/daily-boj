## 💡 백준 문제
- 문제 링크: [백준 5073](https://www.acmicpc.net/problem/5073)
- 문제 요약: 세 변의 길이를 입력받아 삼각형이 되는지 확인하고, 가능하면 종류(Equilateral, Isosceles, Scalene)를, 불가능하면 Invalid를 출력


## 📌 풀이 방법
1. 입력 받은 후 max를 찾음
2. max가 a, b, c 중 어떤건 지 알 수 없으니까 a + b + c - max로 삼각형의 조건을 확인함

## ✅ 느낀 점
Math.max를 a, b만 넣을 생각을 했었는데 Math.max 안에 Math.max를 넣어서 3개 중 max 값을 쉽게 구할 수 있었다.
