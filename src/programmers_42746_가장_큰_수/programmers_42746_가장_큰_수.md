## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 42746](https://school.programmers.co.kr/learn/courses/30/lessons/42746)
- 문제 요약: 주어진 정수들을 이어 붙여 만들 수 있는 가장 큰 수를 구하는 문제.

## 📌 풀이 방법
1. 모든 정수를 문자열로 변환한다. 
2. 두 문자열 a, b에 대해 (b + a)와 (a + b)를 비교하는 기준으로 정렬한다. 
3. 더 큰 결과를 만드는 문자열이 앞에 오도록 내림차순 정렬한다. 
4. 정렬된 문자열을 순서대로 이어 붙인다. 
5. 가장 앞의 값이 0이면 모든 값이 0인 경우이므로 "0"을 반환한다.

## ✅ 느낀 점
숫자 자체의 크기를 비교하는 것이 아니라, 이어 붙였을 때 어떤 순서가 더 큰 수를 만드는지를 비교하는 방법이 중요했다.
