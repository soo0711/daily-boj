## 💡 백준 문제
- 문제 링크: [백준 13909](https://www.acmicpc.net/problem/13909)
- 문제 요약: N개의 창문을 1번부터 N번 사람까지 순서대로 토글했을 때, 마지막에 열려 있는 창문의 개수를 구하는 문제

## 📌 풀이 방법
1. 9까지 직접 풀어보면서 규칙을 찾아냈다.
2. 소수, 제곱수, 합성수 경우를 비교해보며 제곱수일 때만 창문이 열려 있음을 확인했다.
3. 제곱수는 약수가 홀수 개 존재한다.
4. n² ≤ N 인 n의 개수는 n ≤ √N 이므로
5. Math.sqrt를 이용해 정답을 구했다.

## ✅ 느낀 점
눈에 보이는 규칙만으로 판단하면 쉽게 착각할 수 있다는 걸 느꼈다. <br>

여러 경우를 직접 확인해본 결과 <br>
약수의 개수가 홀수가 되는 수는 제곱수뿐이라는 사실이 가장 중요한 포인트였다.
