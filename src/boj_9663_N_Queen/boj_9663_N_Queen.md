## 💡 백준 문제
- 문제 링크: [백준 9663](https://www.acmicpc.net/problem/9663)
- 문제 요약: N×N 체스판에서 N개의 퀸을 서로 공격하지 않도록 배치할 수 있는 모든 경우의 수를 백트래킹으로 구하는 문제

## 📌 풀이 방법
1. 한 행(row)에 하나의 퀸만 놓아야 하고
2. 열(col)과 두 종류의 대각선을 boolean 배열로 관리하여 공격할 수 있는 위치인지 확인한다.
3. 현재 위치에 퀸을 놓을 수 있는 경우에만 재귀 호출을 진행하고 재귀가 끝나면 상태를 원복한다.

## ✅ 느낀 점
처음에는 대각선을 단순히 “옆 칸”이나 “막힌 칸의 개수”로 처리하려다 로직이 계속 꼬였다.
대각선은 개수가 아니라 좌표의 관계(row + col, row - col)로 판단해야 한다는 점이 핵심이라는 것을 이해하고 나서 문제를 훨씬 쉽게 접근할 수 있었다.
이 문제를 통해 백트래킹에서 상태 정의가 얼마나 중요한지를 제대로 느낄 수 있었다.
