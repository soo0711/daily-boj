## 💡 백준 문제
- 문제 링크: [백준 24266](https://www.acmicpc.net/problem/24266)
- 문제 요약: 입력 크기 n이 주어질 때 MenOfPassion 알고리즘에서 코드1`(sum <- sum + A[i] × A[j] × A[k])`의 수행 횟수와 최고차항 차수를 출력해라.

## 📌 풀이 방법
```java
MenOfPassion(A[], n) {
        sum <- 0;
        for i <- 1 to n
            for j <- 1 to n
                for k <- 1 to n
                    sum <- sum + A[i] × A[j] × A[k]; # 코드1
        return sum;
        }
```
1. 삼중 for문은 n^3만큼 실행하기 때문에 수행횟수는 n^3
2. 수행횟수가 n^3이므로 최고차수는 3이다.

## ✅ 느낀 점
삼중중 for문 반복 횟수는 입력 크기에 n^3이다. <br>
시간복잡도는 O(n^3)
