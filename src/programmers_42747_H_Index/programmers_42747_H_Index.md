## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 42747](https://school.programmers.co.kr/learn/courses/30/lessons/42747)
- 문제 요약: 논문 인용 횟수 배열이 주어질 때, h번 이상 인용된 논문이 h편 이상이 되는 최대 h값(H-Index)을 구하는 문제.

## 📌 풀이 방법
1. 인용 횟수 배열을 오름차순으로 정렬한다. 
2. 전체 논문 수를 기준으로 현재 인덱스 i에서 가능한 h값을 h = n - i로 설정한다.
3. citations[i] >= h를 만족하면, 해당 시점에서 h편 이상의 논문이 h번 이상 인용된 것이므로 h를 반환한다.

4. 끝까지 조건을 만족하지 못하면 0을 반환한다.
