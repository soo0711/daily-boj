## 💡 백준 문제
- 문제 링크: [백준 2108](https://www.acmicpc.net/problem/2108)
- 문제 요약: 홀수 개의 정수가 주어질 때 산술평균, 중앙값, 최빈값(여러 개면 두 번째로 작은 값), 범위를 계산하는 문제


## 📌 풀이 방법
1. 산술평균은 N개의 수의 합을 구한 뒤 `Math.round()`를 사용해 반올림하여 출력했다.
2. 중앙값은 배열을 오름차순으로 정렬한 후 가운데 인덱스의 값을 출력했다.
3. 최빈값은 `Map`으로 빈도를 계산하고 빈도 내림차순 → 값 오름차순 기준으로 정렬하여 최빈값이 여러 개인 경우 두 번째 값을 선택했다.
4. 범위는 정렬된 배열의 최댓값과 최솟값의 차를 구해 출력했다.

## ✅ 느낀 점
최빈값이 여러 개인 경우를 처리하는 조건이 핵심이었고 정렬 기준을 명확히 정의하면 코드가 훨씬 단순해진다는 점을 배웠다.
