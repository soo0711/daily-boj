## 💡 백준 문제
- 문제 링크: [백준 13241](https://www.acmicpc.net/problem/13241)
- 문제 요약: 두 정수 A와 B가 주어질 때 최대공약수를 이용해 최소공배수(LCM)를 구하는 문제


## 📌 풀이 방법
1. 1934번 최소공배수 문제와 동일한 방식으로 접근했다.
2. 최소공배수 공식은 `a * b / 최대공약수`이기 때문에 먼저 최대공약수를 구했다. 
3. 최대공약수는 유클리드 호제법을 사용해 구했다.
4. A와 B의 범위가 최대 10^8까지여서 A × B 연산 시 int 범위를 초과할 수 있어 long 타입으로 변환해 overflow를 방지했다.

## ✅ 느낀 점
문제의 입력 범위를 정확히 확인하는 것이 중요하다는 것을 다시 한 번 느꼈다. 
자료형을 신경 쓰지 않으면 쉽게 overflow가 발생할 수 있는 문제였다.
