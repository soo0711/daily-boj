## 💡 백준 문제
- 문제 링크: [백준 10816](https://www.acmicpc.net/problem/10816)
- 문제 요약: 상근이가 가진 숫자 카드들에서 각 질의 숫자가 몇 장씩 있는지 세어 출력하는 문제

## 📌 풀이 방법
1. Map의 map.getOrDefalut()를 사용해서 상근이 카드를 입력해서 중복된 숫자의 개수를 넣어줬다.
2. 질의 숫자카드를 key 값으로 사용해서 value를 출력해줬다.

## ✅ 느낀 점
상근이 카드를 입력할 때 숫자를 세니까 질의 카드에서는 map.getOrDefalut()를 사용해서 key값이 없으면 0 있으면 value값을 출력하면 됐다.
Map의 key값을 사용해 value를 쉽게 얻을 수 있었다.
