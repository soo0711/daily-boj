## 💡 백준 문제
- 문제 링크: [백준 11478](https://www.acmicpc.net/problem/11478)
- 문제 요약: 문자열 S에서 길이 1 이상인 모든 연속 부분 문자열 중 서로 다른 것의 개수를 구하는 문제

## 📌 풀이 방법
1. 서로 다른 것의 개수를 구하는 문제라서 `Set`을 이용해 중복을 없애야겠다고 생각했다.
2. 문자열 S를 `substring`하여 `Set`에 넣어줬다.

## ✅ 느낀 점
문제 자체는 단순했지만, 문자열 인덱스 처리와 반복문 구조를 가독성 좋게 작성하는 것이 얼마나 중요한지 느낄 수 있었다.
