## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 42586](https://school.programmers.co.kr/learn/courses/30/lessons/42586)
- 문제 요약: 진도와 속도가 다른 기능들을 앞선 기능의 완료 시점을 기준으로 묶어 순차 배포할 때, 각 배포마다 몇 개의 기능이 함께 배포되는지 구하는 문제.

## 📌 풀이 방법
1. 각 기능을 큐에 넣고, 먼저 배포되어야 하는 기능부터 처리한다. 
2. 맨 앞 기능이 완료되기까지 걸리는 일수를 계산한 뒤, 해당 기간 동안 뒤의 기능들도 함께 진행된다고 본다. 
3. 같은 기간 내에 완료되는 기능들을 큐에서 연속으로 꺼내며 개수를 센다. 
4. 한 번에 배포되는 기능 수를 리스트에 담고, 이를 배열로 변환해 반환한다.

## ✅ 느낀 점
배포 기준을 앞 기능에 맞춘다는 조건이 중요했다.
