## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 76502](https://school.programmers.co.kr/learn/courses/30/lessons/76502)
- 문제 요약: 문자열을 왼쪽으로 회전시키며 각 경우마다 스택을 이용해 올바른 괄호 문자열인지 검사하고, 그 개수를 구하는 문제.

## 📌 풀이 방법
1. 문자열 길이만큼 반복하면서 각 회전 케이스를 만든다.
2. (i + j) % n 인덱스를 이용해 문자열을 왼쪽으로 회전한 새로운 문자열을 생성한다. 
3. 생성된 문자열을 스택을 활용해 올바른 괄호 문자열인지 검증한다. 
4. 여는 괄호는 스택에 넣고, 닫는 괄호가 나오면 스택의 top과 짝이 맞는지 확인 후 pop한다. 
5. 중간에 짝이 맞지 않거나 스택이 비어있으면 false를 반환한다. 
6. 문자열 검사가 끝났을 때 스택이 비어있으면 올바른 괄호 문자열로 판단한다. 
7. 올바른 경우의 개수를 누적하여 최종적으로 반환한다.

## ✅ 느낀 점
문자열 회전은 직접 substring으로 나누기보다 (i + j) % n 방식으로 처리하니 훨씬 간단하게 구현할 수 있었다.
전체 시간복잡도는 O(n²)이지만 n이 1000 이하라 충분히 통과 가능했다.
