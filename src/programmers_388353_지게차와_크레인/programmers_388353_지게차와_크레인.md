## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 388353](https://school.programmers.co.kr/learn/courses/30/lessons/388353)
- 문제 요약: 지게차와 크레인 요청을 순서대로 처리한 뒤 남아 있는 컨테이너의 개수를 구하는 문제.

## 📌 풀이 방법
1. 외부 접근을 쉽게 확인하기 위해 storage 주변에 빈 공간 테두리를 추가한다. 
2. 요청 문자열의 길이가 2이면 크레인 요청이므로 해당 알파벳 컨테이너를 모두 제거한다. 
3. 요청 문자열의 길이가 1이면 지게차 요청이므로 외부 빈 공간에서 BFS를 시작한다. 
4. BFS 중 빈 공간은 계속 이동하고, 요청한 컨테이너를 만나면 제거 대상에 저장한다. 
5. 같은 요청에서 제거 대상 컨테이너를 모두 찾은 뒤 한 번에 빈 공간으로 바꾼다. 
6. 모든 요청을 처리한 뒤 남아 있는 컨테이너 개수를 세어 반환한다.

## ✅ 느낀 점
외부에서 접근 가능한 빈 공간을 기준으로 BFS를 돌리면 지게차로 꺼낼 수 있는 컨테이너만 구분할 수 있었다.
