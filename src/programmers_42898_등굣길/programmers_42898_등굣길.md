## 💡 프로그래머스 문제
- 문제 링크: [프로그래머스 42898](https://school.programmers.co.kr/learn/courses/30/lessons/42898)
- 문제 요약: 오른쪽과 아래쪽으로만 이동할 수 있을 때, 웅덩이를 피해 집에서 학교까지 가는 최단경로의 개수를 구하는 문제.

## 📌 풀이 방법
1. dp 배열을 만들고 웅덩이 위치를 -1로 표시한다. 
2. 시작점인 dp[1][1]을 1로 초기화한다. 
3. 각 칸마다 위쪽에서 오는 경우와 왼쪽에서 오는 경우를 더한다. 
4. 웅덩이를 만나면 0으로 바꾸고 경로 계산에서 제외한다. 
5. 모든 값은 1,000,000,007로 나눈 나머지로 저장한다. 
6. 마지막 칸 dp[n][m] 값을 정답으로 반환한다.

## ✅ 느낀 점
경로의 개수를 직접 세는 것이 아니라 각 칸까지 도달하는 경우의 수를 누적 해서 계산하는 방식이 효율적이었다.
