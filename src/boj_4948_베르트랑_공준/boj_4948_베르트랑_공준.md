## 💡 백준 문제
- 문제 링크: [백준 4948](https://www.acmicpc.net/problem/4948)
- 문제 요약: 자연수 n이 주어질 때, n < p ≤ 2n 범위에 존재하는 소수의 개수를 구하는 문제

## 📌 풀이 방법
1. 에라토스테네스의 체를 사용해 소수를 미리 구했다.
2. n의 최대값이 123,456이므로, 2n의 최댓값인 246,912까지 한 번만 체를 적용했다.
3. 이후 각 입력마다 (n, 2n] 범위를 순회하며 소수의 개수를 세어 출력했다.

## ✅ 느낀 점
에라토스테네스의 체는 여러 범위에서 소수 개수를 반복적으로 구해야 하는 문제에서  
미리 계산한 결과를 재사용할 수 있어 매우 효율적인 알고리즘이라는 것을 느꼈다.
